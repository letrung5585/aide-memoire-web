# Aide-mémoire Perspective: Universal Problem-Solving Framework

**Aide-mémoire Perspective** is a sophisticated cognitive system designed to transform how we learn and apply knowledge. Moving beyond rote memorization, this platform structures information through a multi-dimensional lens: starting from a real-world challenge, establishing a theoretical solution, and expanding into historical, modern, and creative applications.

## 🧠 The 5 Core Productivity Pillars
To ensure maximum retention and efficiency, the framework integrates five world-class learning and management methodologies:
* **Active Recall:** Interactive flashcards to stimulate memory retrieval.
* **Feynman Technique:** Simplifying complex theories to ensure a deep, fundamental understanding.
* **Spaced Repetition:** Scientifically scheduled review cycles to combat the forgetting curve.
* **Pomodoro Technique:** Time-blocked focus sessions to maintain high mental performance.
* **Eisenhower Matrix:** Smart prioritization of tasks based on urgency and importance.

## 🔄 The Perspective Workflow
Every module follows a consistent logic to bridge the gap between abstract theory and practical mastery:

1.  **The Challenge (Problem Statement):** Defines a concrete, real-world obstacle that requires a solution.
2.  **The Theoretical Solution:** Provides the core principles, formulas, or logic used to address the problem.
3.  **Multi-Dimensional Execution:**
    * 🏺 **Historical Application:** How this problem was solved in the past or its classical origins.
    * 🤖 **Modern Application:** How the theory is utilized in current industry, technology, and daily life.
    * 🎋 **Creative & Flexible Adaptation:** Empowering anyone to adapt the knowledge for diverse problems, DIY projects, or personal innovations.

## 🛠 Technical Excellence
* **Universal Content Engine:** Decoupled architecture allows you to swap entire knowledge domains (Science, Economics, Philosophy, Tech) simply by updating a JSON data file.
* **Intelligent Image Handling:** Automated fallback mechanisms to handle various image formats (.jpg, .png) or online assets, ensuring a seamless visual experience.
* **Responsive UI:** A 3D-flip card interface optimized for both mobile and desktop "Perspective Thinking".
